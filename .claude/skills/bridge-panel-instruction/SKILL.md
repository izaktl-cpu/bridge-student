# כיתוב הוראות על פנל ההוראות (bridge-student)

הפנל הכחול מציג את **ההוראה הכללית של כל הכרזה** — מה התלמיד אמור להכריז ובאילו תנאים.
הסקיל הזה קובע איך לנסח את הטקסט כך שיהיה נקי, בלי סלט של עברית+אנגלית, ובלי מספרים שבורים.

## 1. איך הפנל מרנדר — אל תשתמש ב-fix()

הפנל כבר מפצל כל שורה לאסימונים (`utils/rtl.tokens`) ומרנדר כל אסימון ב-`tk.Label`
נפרד עם `pack(side='right')`. זה מה שמבטל את הסלט. לכן:

- **כתוב טקסט רגיל.** אל תעטוף ב-`fix()` / `fix_num()` ואל תוסיף תווי-בקרה (RLM/LRE/LRO).
- אשכול לטיני — מספר, טווח, הכרזה, סמל חליפה — נשמר **שלם** אוטומטית. `8-9` לא יתהפך ל-`9-8`.
- כל הכרזה יושבת ב-Label משלה, לכן מותר לכתוב עברית והכרזה באותה שורה (`2NT` ליד `8-9 נקודות`).

**אם מוסיפים מקום חדש שמרנדר טקסט לפנל — חובה לעבור דרך `_render_rtl` / `tokens`, לא `fix`.**

## 2. כללי עברית נקייה (חובה)

| אסור | במקום |
|---|---|
| `?` — גם בסוף `מה תכריז?` (מתהפך ב-RTL) | `מה תכריז` בלי סימן שאלה |
| `:` נקודתיים | לשבור לשורה חדשה |
| `—` מקף ארוך | פסיק או שורה חדשה |
| `=` ו-`+` בין מספרים (`13+12=25`) | לנסח במילים |
| `( )` סוגריים בכותרת הוראה | בלי סוגריים |
| `נק׳` / `נק'` | תמיד `נקודות` במלואן |
| `טעית` / נזיפה | הפנל מציג רק את התנאי וההכרזה, בלי נזיפה |
| `מפתחות` (RKCB) | `אסים` — ה-K בשליט נספר כאס |

**מותר:** פסיקים, נקודה בסוף שורה, `סה״כ`, ו-`—` כסימן "אין הכרזה" בשיעור הדאבל בלבד.

## 3. מספרים והכרזות

כתוב אותם בצורתם הטבעית — הטוקנייזר שומר אותם שלמים:

- טווח נקודות: `8-9`, `10+`, `12-14`
- חלוקה: `5-3-3-2`, `4-3-3-3`
- הכרזה: ספרה+סמל/`NT` — `1NT`, `2♣`, `3♥`, `4♠`, `Pass`, `פס`
- כמות קלפים: `5 קלפי ♠`, `6+ קלפי ♥`

הימנע מלהצמיד ספרה לעברית בלי רווח. עדיף `יש 13 נקודות` על פני צירוף דחוס.

## 4. ה-API

**הוראה חופשית** (טקסט קצר, כל משפט בשורה):
```python
self.app.set_instruction('ספור את הנקודות שלך\nיד מאוזנת\nמה תפתח')
```

**טבלת הכרזה + תנאי** — כותרת ורשימת `(הכרזה, תנאי)`:
```python
self.app.set_instruction_table(
    'מחשב פתח 1NT. מה תענה',
    [
        ('פס',  '0-7 נקודות'),
        ('2NT', '8-9 נקודות'),
        ('3NT', '10+ נקודות'),
    ]
)
```
הפנל מציג את ההכרזה מימין ואת התנאי לצידה, כל אסימון נקי ובמקומו.

## 5. דוגמאות טוב / רע

```
❌  פתח 1NT? (12-14 נק׳)          →  ✅  פתח 1NT עם 12-14 נקודות
❌  יש לך 5 קלפי ♠ ו8 נק': מה?    →  ✅  יש 5 קלפי ♠ ו-8 נקודות\nמה תכריז
❌  תגובה = 8+ מפתחות             →  ✅  ענה עם 8+ נקודות
```

## כלל אצבע

טקסט הפנל = **מה התלמיד צריך להכריז ובאיזה תנאי**, בעברית נקייה, כל הכרזה בשורה או ב-Label משלה,
מספרים וטווחים בצורתם הטבעית — בלי `fix()` ובלי סימני פיסוק ששוברים RTL.
